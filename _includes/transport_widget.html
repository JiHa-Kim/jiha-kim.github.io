Monge transport requires each source atom to go to exactly one destination. If the money already comes in the right physical bills, shipping works. If one indivisible bill would have to split across two addresses, Monge fails.

{% endcapture %} {% capture visuals %}
Success
Shipping Two Separate $1 Bills

Each address needs $1, and the source already comes as two separate bills.

Bill Bill A $1, ship intact
Bill Bill B $1, ship intact
A → C
B → D
Address C needs $1
Address D needs $1
A deterministic one-to-one assignment exists.
Failure
Shipping One Indivisible $2 Bill

Addresses C and D still each need $1, but now the source is one physical bill.

Bill Bill A $2, cannot split
A → C
C gets $2, D gets $0
A → D
C gets $0, D gets $2
Address C needs $1
Address D needs $1
No deterministic map can turn one bill into two separate $1 deliveries.
